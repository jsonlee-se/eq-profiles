# eq-profiles

I just wanted a place to keep all my eq profiles across machines.
If you ever stumble on this Repo, DO NOT use these profiles for you own IEMs or Headphones.
* Every profile is meant for the IEM/ Headphone mentioned in the name.
* NOT to try and reproduce that product's sound.
* Is customized to present a close to neutral sound reproduction FOR MY HEAD AND EARS.

These profiles are used with the EQ APO software.
